## 避坑第三步：手把手Java示例——从配置到调用

这才是大多数人最关心的部分。下面直接给一个完整的Java代码示例，帮你在几分钟内接上o3。

**第一步：添加依赖（Maven）**

到你项目的pom.xml中加入OpenAI的Java SDK：

xml
<dependency>
    <groupId>com.theokanning.openai-gpt3-java</groupId>
    <artifactId>service</artifactId>
    <version>0.18.2</version>
</dependency>


**第二步：写调用代码**

创建一个Java类，比如 `O3Connection.java`，内容如下：

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.completion.chat.ChatCompletionResult;
import java.util.List;

public class O3Connection {
    public static void main(String[] args) {
        // 核心改动：把base_url改成云雾的地址
        String token = "你的API密钥"; // 从云雾后台获取
        OpenAiService service = new OpenAiService(token, Duration.ofSeconds(60));
        // 设置自定义base url
        service.setBaseUrl("https://www.yunwuai.cc/v1");

        // 构建请求，模型名改成 o3-full或 o3-mini
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("o3-full")
                .messages(List.of(
                        new ChatMessage("system", "你是一个有用的助手。"),
                        new ChatMessage("user", "请用中文写一段关于AI发展的200字总结。")
                ))
                .build();

        ChatCompletionResult result = service.createChatCompletion(request);
        // 输出返回结果
        result.getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });
    }
}


**第三步：替换参数，运行**

- `token`：替换成你在[云雾ai聚合站](https://www.yunwuai.cc/)后台申请的API密钥。
- `model`：可以填 `o3-full`（o3正式版）或 `o3-mini`（轻量版）。
- `setBaseUrl`：必须改成 `https://www.yunwuai.cc/v1`。

完成。你没有翻墙，没有绑海外卡，只改了3行代码，就成功调通了o3模型。
