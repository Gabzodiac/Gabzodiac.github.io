## 第三步：编写你的第一段 Mistral 调用代码

python
from openai import OpenAI

# 1. 创建一个客户端，核心就是改这两个参数
client = OpenAI(
    api_key="这里填你刚才复制的 API Key sk-xxxx",

    max_tokens=500    # 控制最大输出字数

# 3. 输出模型的回答

**代码逐行解析：**

- `from openai import OpenAI`: 引入我们刚才安装的库。
- `api_key="..."`: 这里填你在云雾API控制台里获得的那个 `sk-` 开头的密钥。
- `base_url="https://www.yunwuai.cc/v1"`: **这是最核心的一步**。你不需要去查 Mistral 官方是什么 API 地址，直接用云雾提供的这个地址，它就帮你转接好了。

- `messages`: 这是 API 传入的对话列表。`role` 是 `user` 表示你是用户，`assistant` 表示 AI 回答。
- `temperature`: 数值越大回答越有创造性，数值越小回答越严谨。
- `max_tokens`: 限制回答的最大长度，防止输出太长。

## 第四步：出错怎么办？常见问题排雷

新手常见报错无非就几种，下面直接给你解决方案：

**报错 1：`AuthenticationError: Incorrect API key provided`**

原因：API Key 抄错了，或者 Key 前面带了空格/换行符。

解决：重新打开控制台，复制你的 Key，确保粘贴后两边没有多余字符。

**报错 2：`APIConnectionError` 或 `TimeoutError`**

原因：网络问题，或者 API 地址写错了。

解决：检查你的 `base_url` 是否写成了 `https://www.yunwuai.cc/v1`，注意是 `https` 不是 `http`。如果在公司网络，需要检查是否有防火墙拦截。

**报错 3：`NotFoundError`**

原因：模型名字写错了，或者该模型在平台上不可用。

解决：你可以去云雾API的官方文档查看当前支持的最新模型列表。常用的 Mistral 模型还有 `mistralai/Mixtral-8x7B-Instruct-v0.1`、`mistralai/Mistral-Large-Instruct-2407` 等。

## 第五步：进阶技巧——流式输出与多轮对话

**流式输出（让文字像打字一样逐字出现）**

上面的代码会等模型全部回答完才一次性输出。如果你想获得类似 ChatGPT 那样的实时打字效果，只需要改一点点代码：

python
import time  # 为了演示效果，加上时间延迟

from openai import OpenAI

client = OpenAI(
    api_key="你的 Key",
    base_url="https://www.yunwuai.cc/v1"
)

# 区别在于 stream=True
stream = client.chat.completions.create(
    model="mistralai/Mistral-7B-Instruct-v0.2",
    messages=[
        {"role": "user", "content": "用一句话解释量子纠缠"}
    ],
    stream=True  # 启用流式输出
)

# 逐块打印
for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")
        time.sleep(0.02)  # 让你的眼睛跟上打字速度


这样运行后，文字就会像实时打字一样在终端里逐字出现了。

**多轮对话（让 AI 记住上下文）**

在 `messages` 列表里加入助手的回复，就能实现连续对话：

python
messages = [
    {"role": "system", "content": "你是一个中文诗歌助手，回答要简洁、有诗意。"},
    {"role": "user", "content": "写一句关于秋天的五言诗"},
    {"role": "assistant", "content": "落叶舞秋风"},
    {"role": "user", "content": "再续一句，要更加悲凉"}  # 这是本轮对话
]

response = client.chat.completions.create(
    model="mistralai/Mistral-7B-Instruct-v0.2",
    messages=messages
)

print(response.choices[0].message.content)


几乎所有现代模型都支持这种通过 `messages` 列表传递历史对话的方式来保持上下文。
