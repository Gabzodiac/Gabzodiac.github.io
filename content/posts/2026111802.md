## 手把手：让程序"看"图片

### 第一步：准备图片

Qwen-VL 支持你给两种格式的图片：一种是图片公开的 **URL 链接**，另一种是把图片内容编码成 **Base64 字符串**。

为了方便演示，我直接把一张图片本地编码好，模拟真实场景里本地图片的分析需求。

假设你有一张图片在电脑的 `E:/my_image.jpg` 路径下，可以用 Python 的 `base64` 模块把它转成字符串：

import base64

def encode_image_to_base64(image_path):
    with open(image_path, "rb") as image_file:
        return base64.b64encode(image_file.read()).decode('utf-8')

# 使用上面的函数
image_path = "E:/my_image.jpg"  # 改成你的图片路径
base64_image_string = encode_image_to_base64(image_path)
print("图片已编码完成，打印前100个字符：", base64_image_string[:100])

如果你的图片在互联网上，这一步可以省略，直接用图片的 URL 链接。

## 关于图像格式的一点提醒

Qwen-VL 能处理大多数常见的图片格式，包括 `png`、`jpg`、`webp` 和 `gif`（不过动态图只会分析第一帧）。如果你要上传高分辨率图片，Qwen-VL 会自动帮你缩放处理，不用担心调用失败。

图片的 Base64 编码在 Python 里很好做，但如果图片太大（比如几兆以上），建议还是用公开 URL 或者本地压缩尺寸，这样编码后的字符串会短很多，传输也更快。
