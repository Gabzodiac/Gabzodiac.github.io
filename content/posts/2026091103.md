## 流式调用（Streming）示例：适合大模型实时对话场景

如果你希望调用 o3-mini 时，能像打字机一样逐字显示回答，用流式调用最合适。[云雾api中转站](https://www.yunwuai.cc/)完美支持流式输出。

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import io.reactivex.Flowable;
import java.util.List;

public class O3MiniStreamDemo {

    public static void main(String[] args) throws InterruptedException {
        OpenAiService service = new OpenAiService("sk-你的云雾API密钥");

        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("o3-mini")
                .messages(List.of(
                        new ChatMessage("user", "用Java递归实现一个斐波那契数列。")
                ))
                .maxTokens(1024)
                .stream(true)   // 启用流式
                .build();

        // 流式接收
        Flowable<ChatCompletionChunk> flowable = service.streamChatCompletion(request);
        flowable.blockingForEach(chunk -> {
            String delta = chunk.getChoices().get(0).getMessage().getContent();
            if (delta != null) {
                System.out.print(delta); // 逐个打印 token
            }
        });
    }
}
