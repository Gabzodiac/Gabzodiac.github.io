## 高级进阶：流式输出与异步调用

当你跑通上面的基础示例后，可以尝试流式输出（Streaming），让模型像打字一样逐词输出，用户体验更好。

python
completion = client.chat.completions.create(
    model="Qwen/Qwen3-8B",
    messages=[...],  # 同上
    stream=True,  # 关键参数
)

for chunk in completion:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="")
