## 从零开始：Java 项目接入步骤

下面就是完整的 Java  Spring Boot 项目集成示例了。我用的是最标准的 OpenAI Java 客户端。

### 第一步：创建 Maven 项目

在 `pom.xml` 中添加依赖：

xml
<dependency>
    <groupId>com.theokanning.openai-gpt3-java</groupId>
    <artifactId>service</artifactId>
    <version>0.18.2</version>
</dependency>


这个库默认兼容 OpenAI 接口格式，而云雾完全兼容这个标准，所以可以直接用。

### 第二步：创建配置类

在 `application.yml` 中添加云雾配置：

yaml
ai:
  api:
    base-url: https://www.yunwuai.cc/v1
    api-key: 你的_API_KEY
    model: ernie-4.0-8k


注意这里的 `base-url` 必须写成云雾的 API 接口地址：`https://www.yunwuai.cc/v1`，`model` 字段直接写文心一言的模型名。

### 第三步：编写调用服务

创建一个 `AiService` 类：

java
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.completion.chat.ChatCompletionResult;
import com.theokanning.openai.service.OpenAiService;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Service;

import java.time.Duration;
import java.util.List;

@Service
public class AiService {

    @Value("${ai.api.api-key}")
    private String apiKey;

    @Value("${ai.api.base-url}")
    private String baseUrl;

    @Value("${ai.api.model}")
    private String model;

    public String chat(String userMessage) {
        // 构建 OpenAiService 实例，传入云雾的 baseUrl
        OpenAiService service = new OpenAiService(apiKey, Duration.ofSeconds(60));
        service.setBaseUrl(baseUrl);

        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model(model)
                .messages(List.of(
                        new ChatMessage("system", "你是一个乐于助人的助手。"),
                        new ChatMessage("user", userMessage)
                ))
                .maxTokens(2000)
                .temperature(0.7)
                .build();

        ChatCompletionResult result = service.createChatCompletion(request);
        return result.getChoices().get(0).getMessage().getContent();
    }
}


关键点只有两个：
- 在 `new OpenAiService` 之后，调用 `setBaseUrl(baseUrl)` 设置自定义地址
- 其他所有代码都和你调 OpenAI 时一模一样

### 第四步：创建测试类
