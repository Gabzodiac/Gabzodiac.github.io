如果返回正常，说明连接成功。

### 2. 错误码排查

- **401 Unauthorized**：检查 API Key 是否正确，或是否已过期
- **404 Model Not Found**：确认模型名写的是 `gemini-2.0-flash-001`，不是 `gemini-2.0-flash`
- **429 Too Many Requests**：触发限流，适当增加请求间隔
- **500 Internal Server Error**：通常是临时问题，重试一次即可

### 3. 调试参数优化

- **`max_tokens`**：控制输出长度，Gemini 2.0 Flash 支持最多 4096 tokens
- **`temperature`**：控制随机性，代码生成推荐 0.2，创意任务推荐 0.7
- **`top_p`**：采样优化，一般和 temperature 配合使用
