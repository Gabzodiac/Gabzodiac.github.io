## 真实接入体验：Java代码就这么简单

说再多不如看代码。以下是使用云雾官网Java封装库接入DeepSeek-R1的完整示例：

java
import okhttp3.*;
import org.json.JSONObject;
import java.io.IOException;

public class DeepSeekDemo {
    static final String API_URL = "https://www.yunwuai.cc/v1/chat/completions";
    static final String API_KEY = "你的云雾API Key"; // 在云雾官网注册后获取

    public static void main(String[] args) throws IOException {
        OkHttpClient client = new OkHttpClient();

        JSONObject requestBody = new JSONObject();
        requestBody.put("model", "deepseek-r1");
        JSONObject message = new JSONObject();
        message.put("role", "user");
        message.put("content", "用Java写一个快速排序算法，并解释复杂度。");
        requestBody.put("messages", new JSONArray().put(message));

        Request request = new Request.Builder()
            .url(API_URL)
            .addHeader("Authorization", "Bearer " + API_KEY)
            .addHeader("Content-Type", "application/json")
            .post(RequestBody.create(
                MediaType.parse("application/json; charset=utf-8"),
                requestBody.toString()
            ))
            .build();

        try (Response response = client.newCall(request).execute()) {
            if (response.isSuccessful()) {
                String responseBody = response.body().string();
                JSONObject json = new JSONObject(responseBody);
                String content = json.getJSONArray("choices")
                    .getJSONObject(0)
                    .getJSONObject("message")
                    .getString("content");
                System.out.println("DeepSeek回答：" + content);
            } else {
                System.out.println("请求失败：" + response.code());
            }
        }
    }
}


这段代码里面，唯一需要你动手改的只有两行：一是`API_KEY`换成你在云雾官网注册后申请的key，二是`model`字段指定你想要的模型名称。其他部分和调用OpenAI API一模一样，直接复制就能用。
