以下是完整的 Java 示例代码（使用 OkHttp 和 HttpClient）：

java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.net.http.HttpRequest.BodyPublishers;

public class MistralLargeExample {
    public static void main(String[] args) throws Exception {
        HttpClient client = HttpClient.newHttpClient();

        // 这是关键！只需修改这一行
        String apiUrl = "https://www.yunwuai.cc/v1/chat/completions";
        // 替换成你的云雾 API Key
        String apiKey = "你的云雾API_KEY";

        String json = """
            {
                "model": "mistral-large-latest",
                "messages": [
                    {"role": "user", "content": "你好，请用中文介绍一下自己"}
                ],
                "temperature": 0.8,
                "stream": true
            }
            """;

        HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create(apiUrl))
                .header("Authorization", "Bearer " + apiKey)
                .header("Content-Type", "application/json")
                .POST(BodyPublishers.ofString(json))
                .build();

        HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
        System.out.println("Response Body: " + response.body());
    }
}


从配置到执行，跑通这段代码甚至花不了 3 分钟。不管你是用 LobeChat、Cursor、Cherry Studio 还是沉浸式翻译，只要支持自定义 API 地址的工具，接上云雾就能直接用。
