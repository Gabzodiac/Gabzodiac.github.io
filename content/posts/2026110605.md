创建一个新的Java类，例如叫 `DeepSeekDemo.java`，把下面代码复制进去：

java
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.io.OutputStream;
import java.net.HttpURLConnection;
import java.net.URL;

public class DeepSeekDemo {
    public static void main(String[] args) throws Exception {

        // 1. 配置API Key和模型地址
        // ⚠️ 注意：把你的 API Key 替换到这里
        String apiKey = "sk-你的api密钥";
        // ⚠️ 核心：兼容OpenAI格式接口，这里直接从你的Java代码怼过去
        String urlStr = "https://www.yunwuai.cc/v1/chat/completions";

        // 2. 构建请求体（这里就是你要问AI的话）
        String prompt = "你好，你是谁？";
        String body = "{"
                + "\"model\": \"deepseek-chat\","
                + "\"messages\": [{\"role\": \"user\", \"content\": \"" + prompt + "\"}],"
                + "\"temperature\": 0.7"
                + "}";

        // 3. 准备工作：建立连接
        URL url = new URL(urlStr);
        HttpURLConnection conn = (HttpURLConnection) url.openConnection();
        conn.setRequestMethod("POST");
        conn.setRequestProperty("Authorization", "Bearer " + apiKey);
        conn.setRequestProperty("Content-Type", "application/json");
        conn.setDoOutput(true);

        // 4. 发送请求
        OutputStream os = conn.getOutputStream();
        os.write(body.getBytes());
        os.flush();
        os.close();

        // 5. 读取响应
        BufferedReader reader;
        if (conn.getResponseCode() == 200) {
            reader = new BufferedReader(new InputStreamReader(conn.getInputStream()));
        } else {
            reader = new BufferedReader(new InputStreamReader(conn.getErrorStream()));
        }
        StringBuilder response = new StringBuilder();
        String line;
        while ((line = reader.readLine()) != null) {
            response.append(line);
        }
        reader.close();

        // 6. 掏出结果
        System.out.println("用户：" + prompt);
        // 简单结果处理，一般包含在 choices[0].message.content 里
        String result = response.toString();
        // 提取AI回复
        int startIdx = result.indexOf("\"content\":\"") + 11;
        int endIdx = result.indexOf("\"", startIdx);
        if (startIdx > 10 && endIdx > startIdx) {
            String aiReply = result.substring(startIdx, endIdx);
            System.out.println("AI：" + aiReply);
        } else {
            System.out.println("AI回复解析失败，请查看原始响应：");
            System.out.println(result);
        }
    }
}

## 进阶玩法：这样写代码才叫专业

上面给的是纯JDK版本，适合零基础理解原理。如果你有一定基础，想在公司项目里用得更优雅，我推荐你用 `OkHttp`。

**Maven依赖（加到pom.xml）：**
xml
<dependency>
    <groupId>com.squareup.okhttp3</groupId>
    <artifactId>okhttp</artifactId>
    <version>4.12.0</version>
</dependency>


**完整代码：**
java
import okhttp3.*;

public class DeepSeekOkHttp {
    public static void main(String[] args) throws Exception {
        OkHttpClient client = new OkHttpClient.Builder()
                .connectTimeout(30, java.util.concurrent.TimeUnit.SECONDS)
                .build();

        String json = "{"
                + "\"model\": \"deepseek-chat\","
                + "\"messages\": [{\"role\": \"user\", \"content\": \"你好，我叫小白！\"}]"
                + "}";

        Request request = new Request.Builder()
                .url("https://www.yunwuai.cc/v1/chat/completions")
                .addHeader("Authorization", "Bearer 你的api密钥")
                .addHeader("Content-Type", "application/json")
                .post(RequestBody.create(json, MediaType.parse("application/json")))
                .build();

        Response response = client.newCall(request).execute();
        System.out.println(response.body().string());
    }
}

你看，核心代码就这几行。记住这个核心套路：**把 `base_url` 指向 `https://www.yunwuai.cc/v1` 就万事大吉了**。
