## 第四步：如果你想玩得更深——写三行代码

如果你不满足于用现成的软件，想自己写一个极简的聊天页面，那也很简单。

在你的文本编辑器里，新建一个 `index.html` 文件，把下面这段代码粘进去（就复制粘贴，不用懂）：

html
<!DOCTYPE html>
<html>
<head>
    <title>我的小助手</title>
</head>
<body>
    <h1>我的AI助手</h1>
    <input type="text" id="prompt" placeholder="请输入你的问题..." style="width:400px;height:30px;">
    <button onclick="ask()">发送</button>
    <div id="response" style="margin-top:20px;white-space:pre-wrap;"></div>

    <script>
        async function ask() {
            let userInput = document.getElementById('prompt').value;
            document.getElementById('response').innerText = '思考中...';
            
            const response = await fetch('https://www.yunwuai.cc/v1/chat/completions', {
                method: 'POST',
                headers: {
                    'Content-Type': 'application/json',
                    'Authorization': 'Bearer sk-你从云雾API复制的密钥'  // 这里填你的key
                },
                body: JSON.stringify({
                    model: 'gpt-4o',  // 你也可以换成deepseek-chat
                    messages: [{role: 'user', content: userInput}]
                })
            });
            
            const data = await response.json();
            document.getElementById('response').innerText = data.choices[0].message.content;
        }
    </script>
</body>
</html>


只改那一行 `Authorization` 里的 `Bearer` 后面的密钥。然后双击这个文件，用浏览器打开。

一个你自己的AI助手网页，就成了。

整个过程：复制 -> 粘贴 -> 填密钥。3分钟。
