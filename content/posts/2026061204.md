### 1. OpenAI SDK 接入（标准方案）

这是目前最通用的方案，因为 DeepSeek-R1 的 API 接口设计完全兼容 OpenAI 格式。

python
import openai

client = openai.OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.deepseek.com/v1"
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role": "user", "content": "写一段Python代码快速排序"}],
    stream=False
)

print(response.choices[0].message.content)


**优点**：代码最标准，成熟度高，SDK 版本兼容性好（推荐 openai>=1.0）。

**缺点**：你需要自己找稳定可靠的 DeepSeek API 提供商。直接连官方 DeepSeek 接口，海外节点多，国内延迟高，且账单支付需绑海外卡，非常麻烦。

**成本陷阱**：部分自称“官方 API”的代理商收费并不透明，官方标价 1M 输入 token 约2元，但有些平台的“基础费率”会在背后偷偷加个1.5倍系数，等你跑完批处理账单翻倍。
